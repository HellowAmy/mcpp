# 前言


## 作出你的选择
如果你是一个编程新手，从来没有学过任何编程的，经过一系列思考之后在众多的编程语言中选择了C++语言，那么恭喜你，你选择了一个又古老又复杂又落后又麻烦的编程语言  
C++是一门古老的编程语言，在历史长河中能存活下只有一个最关键的优势，性能性能性能，除此之外对比现代编程语言没有任何优势，如果你决定以C++作为入门语言，那将是地狱开局  
C++的缺点有很多，比如没有官方统一的编译方式，没有统一的代码规范，没有项目管理功能，这意味着你需要在项目开始之前先考虑如果构建你的项目，使用什么平台下的什么编译方式，手动的去构建和管理你的项目，简而言之，C++是一种又古老又落后又复杂又麻烦的语言   
正因为如此，C++的工程项目是十分灵活的，因为运行的平台不确定，编译方式不确定，编写规范不确定  
在历史遗留中，C++兼容了多种代码编写风格和设计方式，存在非常灵活的设计空间，会导致在代码编写与设计上千人千面，没有统一规范，新手学习代码没有范本，复杂程度指数上升  
我第一门编程语言选择的是C++，且长期以来主要使用C++进行开发，在此之后也借助AI学习了多种编程语言，在使用其他现代编程语言之后感叹，C++太落后了，非常不建议作为新手的第一门编程语言  
这是一段新手劝退的话，但如果你不是新手，也已经掌握至少一种编程语言，那你很清除我在说什么  


## AI时代下的编程
当然你有不得不选的理由，这个教程系列面对的没有任何编程的新手，如果你已经是老鸟，多学学几种语言是很好的事情  
在AI时代下，如果你没有任何编程经验，AI可以帮助你掌握编程知识，所以会使用AI是现代学习编程语言的重要手段，AI可以帮助你学习更深层次的技术，而我只是负责引导你，并给你建立良好的 **编程规范**


## 规范你的行为
作为你的第一门语言，你必然没有规范可言，而大多数的教程都注重掌握技术与技巧，AI也只能从片面的代码中掺杂者一些其他使用者的编程规范，者会导致一部分人从新手入门到进入工作与他人合作，都没有建立起属于自己的一套良好规范，导致合作者的抱怨与鄙视  
我会帮助你建立一个良好的编程规范， **编程规范** ，没错就是编程规范，在C++中编程规范非常重要，是比技术问题都要重要的东西，我会反复强调 **编程规范**，因为C++本身是不安全的，不规范的代码十分容易引发错误且很难排除到问题，编程规范是保持代码一致性的重要手段，在C++中十分重要   
在AI时代，细枝末节的技术技巧可以被暂时忽略，因为你可以通过AI来帮你完成你想要的功能，但编程规范必须要形成自己的风格，形成一套可以自己完全掌控的规则，也被他人认可的规则，保证代码风格一致性  

~~~
项目规范 > 国际规范 > 个人规范
~~~

编程规范与风格是有优先级的，任何人都优先保持与项目的规范同步，保持一致性才能避免产生更多问题，
在学习时，都优先采用国际规范，但和团队合作时，往往项目发起人加入个人规范，这会让每个项目规范都不一样，所以请优先遵守项目规范，一致性才是最重要的  
事实上，国际规范是大多数人的习惯，这何尝不是一种个人规范，所以我会自己的个人规范引导你进入C++入门阶段

## 跨出舒适区
学会使用Google很重要，正如学习编程，学会看懂英文文档比学会编程语法要重要的多，进入Google浏览器页面或许也是走出舒适圈的重要一步

学习C++与Linux系统完全可以同步进行，我相信这些问题AI能帮助到你
